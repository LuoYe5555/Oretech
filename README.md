# Oretech（矿物科技）
![Rrhar'il](https://github.com/LuoYe5555/Oretech/assets/142316619/0eeafa55-e7dc-492e-a940-4a556e2a5da2)

- 矿物科技是用自定义粘液做出的自定义附属:
- 旨在让矿物不只可以用于普通的物品或者简单的机器
- 却还可以做成机器或生成器，制造更多的矿物
- 避免矿物枯竭的状态，且比无尽科技更加便宜，也更加快速
## 前置插件
- Slimefun
- GuizhanLibPlugin
- SlimeCustomizer
 ## 版本
- Mc：1.18.2+
- SlimeCustomizer：≥26版本
